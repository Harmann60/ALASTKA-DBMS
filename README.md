# ALASTKA – Country-Based Tourism Information System

ALASTKA is a Database Management System (DBMS) project designed to store, manage, and retrieve tourism-related information across different countries and cities.

The name **ALASTKA** is inspired by Alaska and Chukotka—regions that appear far apart on flat maps but are geographically close—highlighting map projection illusions.

## Features
- Country-wise tourism information
- City and attraction management
- Categorization of tourist places
- Booking and visitor data handling
- Support for ER and EER modeling concepts

## Technologies Used
- MySQL
- SQL
- Draw.io (ER/EER Diagrams)
- GitHub
